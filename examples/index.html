Native Client Examples

This page has quick links to the SDK examples. Click on the sample names to run them.

Hello World in C
The Hello World example combines HTML, JavaScript, and C code (the C is compiled to create a .nexe file). The .nexe code responds to button click events and calls the browser to show alert panels.
Hello World in C++
The Hello World example combines HTML, JavaScript, and C++ (the C++ is compiled to create a .nexe file). The .nexe code responds to button click events and calls the browser to show alert panels.
Get URL
The Get URL example combines HTML, JavaScript, and C++ (the C++ is compiled to create a .nexe file). The .nexe code uses the Pepper API to fetch an URL and then display its contents.
Pi Generator
The Pi Generator example combines HTML, JavaScript, and C++ (the C++ is compiled to create a .nexe file). The .nexe code creates a thread that estimate pi using the Monte Carlo method while randomly putting 1,000,000,000 points inside a 2D square that shares two sides with a quarter circle. The 2D square is generated using the Pepper 2D API.
Sine Wave Synthesizer
The Sine Wave Synthesizer example combines HTML, JavaScript, and C++ (the C++ is compiled to create a .nexe file). It synthesizes and plays a sine wave. You can enter the frequency of the sine wave. It uses Pepper Audio API to play the tone.
Load Progress
The Load Progress example combines HTML, JavaScript, and C++ (the C++ is compiled to create a .nexe file). The HTML and embedded JavaScript show how to listen for and handle events sent while the NaCl module loads. In particular, the ‘progress’ outputs loading progress.
