Jump 'n Bump Native Client Demo

This is a Native Client port of the multiplayer game Jump 'n Bump.

To play this game, you must use Google Chrome version 10.x or greater, and you must enable Native Client (type about:flags in the address bar, click the "Enable" link under Native Client, and restart Google Chrome).

You can control three of the bunnies by using (1) the W-A-S-D keys, (2) the arrow keys, and (3) the left, middle, and right mouse buttons. The fourth bunny is not available.
