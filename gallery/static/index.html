Native Client Examples Gallery


 

This page lists a few simple web applications that use Native Client modules.

For information about how to develop Native Client applications, see the Native Client SDK

Application Screenshot

Hello, World!

This application combines HTML, JavaScript, and a Native Client module written in C++ and compiled into a .nexe file. The Native Client module responds to button click events and calls the browser to display alerts.

Pi Generator

This application combines HTML, JavaScript, and a Native Client module written in C++ and compiled into a .nexe file. The Native Client module creates a thread that estimates the value of pi through a Monte Carlo method that randomly places 1,000,000,000 points inside a square with an overlaid quarter circle. The Native Client module uses the Pepper API to draw the square.

View source

Sine Wave Synthesizer

This application combines HTML, JavaScript, and a Native Client module written in C++ and compiled into a .nexe file. The Native Client module synthesizes audio samples using a sine wave with a user-specified frequency, and uses the Pepper API to play the resulting tone.

View source

Conway's Life

This application combines HTML, JavaScript, and a Native Client module written in C++ and compiled into a .nexe file. The Native Client module implements the “Life” cellular automaton invented by British mathematician John Horton Conway. Users can paint extra cells into the simulation using a mouse.
